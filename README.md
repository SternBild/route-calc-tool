# 路程計算ツール

## 概要

これは、ブラウザ上で動作するインタラクティブな路程計算ツールです。特定の地域の地図データを内蔵しており、地点をクリックするだけで出発地、経由地、到着地を選択し、最適な経路を計算することができます。

## 主な機能

*   **インタラクティブな地図**:
    *   マウスドラッグで地図を移動
    *   マウスホイールで地図を拡大・縮小
*   **簡単な経路設定**:
    *   地図上の地点をクリックして、出発地、到着地、中継地点（最大5つ）を直感的に選択できます。
*   **高度な経路計算**:
    *   最短経路だけでなく、距離に基づいた第二・第三候補の経路も提示します。
    *   「山道を使用しない」オプションで、特定の種類の道路を避けた経路探索が可能です。
*   **結果の可視化**:
    *   計算された経路が地図上にハイライト表示されます。
    *   複数の経路候補を同時に表示するモードも搭載しています。
*   **PDF出力**:
    *   現在の表示内容（地図、選択地点、計算結果）をPDFファイルとして出力する機能を備えています。

## 使い方

1.  特別なインストールは不要です。
2.  `index.html` ファイルをウェブブラウザ（Google Chrome, Firefoxなど）で開くだけでツールが起動します。
3.  地図上の地点をクリックして、出発地・到着地を選択します。
4.  必要に応じて、さらに地点をクリックして経由地を追加します。
5.  「経路を計算」ボタンを押すと、結果が表示されます。

## 技術的な詳細

このツールは、外部ライブラリ（`html2canvas`, `jspdf`）を除き、単一の`index.html`ファイル内にHTML, CSS, JavaScriptがすべて記述されています。地図の描画にはHTML Canvas APIが使用されています。
